Для сборки сервера создан простой Makefile. 
В терминале, в каталоге проекта можно ввести: 
make - для сборки проекта. После сборки проекта будет создан файл server.
make clean - для очистки объектных файлов
make fclean - для полной очистки всех созданный фалов
make re - пересобирает проект

Для запуска сервера необходимо, в каталоге проекта ввести команду make, и из консоли запустить файл server, с передачей в него строки с номером порта. Если номер порта не будет введен, то сервер будет запущен с портом по умолчанию: 28888. Номер порта должен состоять только из цифр, в противном случае, сервер не будет запущен и будет выведено сообщение об ошибке.

После запуска файла server будет запущен запущен сервер на локальном хосте(localhost) и с указанным портом.
